# ipsw kernel syscall

Dump kernelcache syscalls

```
ipsw kernel syscall [flags]
```

## Options

```
  -g, --gen             Generate syscall table data gzip file
  -h, --help            help for syscall
  -o, --output string   Output gzip file
```
