# ipsw idev crash pull

Pull all crashlogs

```
ipsw idev crash pull [flags]
```

## Options

```
  -a, --all             Pull all crashlogs
  -h, --help            help for pull
  -o, --output string   Folder to save screenshot(s)
  -r, --rm              Remove crashlogs after pulling
```
