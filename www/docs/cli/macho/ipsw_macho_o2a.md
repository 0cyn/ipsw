# ipsw macho o2a

Convert MachO offset to address

```
ipsw macho o2a <macho> <offset> [flags]
```

## Options

```
  -a, --arch string   Which architecture to use for fat/universal MachO
  -d, --dec           Return address in decimal
  -h, --help          help for o2a
  -x, --hex           Return address in hexadecimal
```
