# ipsw macho lipo

Extract single MachO out of a universal/fat MachO

```
ipsw macho lipo [flags]
```

## Options

```
  -a, --arch string     Which architecture to use for fat/universal MachO
  -h, --help            help for lipo
      --output string   Directory to extract the MachO
```
